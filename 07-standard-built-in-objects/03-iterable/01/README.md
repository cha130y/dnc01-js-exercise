- จงเขียนฟังก์ชัน generateArray(n, start) เพื่อคืนค่า array ที่มีความยาวเท่ากับ n โดยสมาชิกของ array ที่ index 0 มีค่าเท่ากับ start และสมาชิกลำดับถัดไปมีค่าเพิ่มขึ้นทีละ 1

```js
generateArray(5, 10); // [10, 11, 12, 13, 14]
generateArray(3, 0); // [0, 1, 2]
generateArray(4, -2); // [-2, -1, 0, 1]
```
