- จงเขียนฟังก์ชัน isSubset = (array1, array2) เพื่อตรวจสอบว่าทุกสมาชิกของ array2 เป็นสมาชิกของ array1 หรือไม่
- ถ้าทุกสมาชิกของ array2 เป็นสมาชิกของ array1 จะคืนค่า true ถ้าไม่ทั้งหมดจะคืนค่า false

```js
isSubset([1, 2, 3, 4, 5, 6, 7], [5, 7, 6]); // true
isSubset([1, 2, 3, 4, 5, 6, 7], [5, 8, 7]); // false
```
